Как рассчитывается % бонуса в ИЗ

Каждому сотруднику на ежемесячной основе выставляется индивидуальное задание (ИЗ), результаты выполнения которого отображаются в СКиП.

ИЗ состоит из нескольких целей, каждая из которых имеет определенный вес (общий вес целей должен быть равен 100%).

По результатам % выполнения целей рассчитывается % премии по тренду, исходя из которого рассчитывается % бонуса за месяц (Показать):

% бонуса к начислению рассчитывается по формуле:

% бонуса для М1 = % премии по тренду * Вес цели

% бонуса для М2/М3 рассчитывается по формуле:

% бонуса для М2/М3 = % премии по тренду * Вес цели * 33%

(максимальный бонус М2/М3 составляет 39,6% от оклада)
